- pnpm + vite

## 游戏要求
以下要求仅是我认为必须的部分,但你应从更专业的角度,将项目做成一个完整的游戏,包括并不限于以下:

- 游戏使用单帧静态图片素材,我在public里为你提供了 英雄静态图: public\images\hero.png, 怪物(0到23,一共24张):public\images\monster, 单位统一使用128*128尺寸
- 由于游戏使用静态图,没有序列帧,因此你需要通过practices, animate, shader等实现丰富的法术技能动画效果,设计好看的单位受击,法术击中等等效果, 必要时可使用类icon字符作为粒子
- 角色wasd控制,单位血量,英雄经验等等,
- 丰富的不同法术,法术显示,角色升级后3选1学习新法术或增强已有法术
- 模拟怪物移动的特效
